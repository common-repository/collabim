Collabim widget for WP admin.

== Description ==

Collabim widget is providing most signifficant data about your measured website from the Collabim system directly to your favourite WP admin. You could find there your actual Collabim index, top 10 keywords and domain’s actual positions, most valuable competitors to your domain and backlink problem reporting. Plugin is using your unique API key to send requests to Collabim servers and to get data about your projects back to WP admin.

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add your API key in plugin -> "Nastavení" section
4. Enjoy your data

== Frequently asked questions ==

= Do I need a Collabim account to be able to use the plugin? =

Yes, in order to get the API key you need to have a Collabim account.

= Where to get the API key? =

You can get your API key in user settings menu at https://my.oncollabim.com/. After logging in, go to 'Mé údaje' at a right corner of a page.
